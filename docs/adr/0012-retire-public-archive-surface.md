# 下线公共档案界面并保留内部资料

公共网站以十五个原创城市场景、候选短句和明确标注的 AI 合成试音为当前产品范围。旧资料柜、档案条目、专题文章和档案 API 不再提供页面、导航、站点地图或静态交付入口；旧地址直接返回 404，不建立到城市故事的误导性映射。

历史档案、专题内容、永久编号、来源、权利、修订记录和治理测试继续作为仓库内部研究资料保留。下线公共界面不等于删除研究资料，也不撤销已有编号。若未来重新公开档案，必须重新审核内容范围、权利、隐私、访问性和与城市故事的关系，并通过新的 ADR 决定公开入口。

这一决定取代 ADR 0001 中“城市故事链接旧资料”的公共入口安排、ADR 0009 中对 `/browse` 搜索行为的交付要求，以及 ADR 0010 中“稳定网址和外部引用以编号为核心”的当前公共实现；其余证据、授权、保存和编号治理原则继续有效。

公众参与入口改为“反馈与纠错”，用于短句语境、页面错误、版权、隐私和无障碍问题。D1 中既有线索表名和状态属于内部实现，不要求暴露在界面文案中。候选短句的“本地使用者复核”是对措辞、语境或试音的核对，不替代档案内容的编辑或专家审核状态。

真人语音采集包属于未来采集工作的独立备用工具，不是旧资料柜或当前 AI 试音改进方案。它保留页面和下载，但不进入 Footer 或当前反馈主流程；只读静态镜像仍不携带该工具和下载文件。
