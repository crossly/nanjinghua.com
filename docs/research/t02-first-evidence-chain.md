# T02 首条证据链研究：刘顺《三十年来的南京方言研究》

研究日期：2026-07-17  
用途：为 `NJH000001` 和一篇短专题文章提供一条范围窄、可核查、权利边界清楚的首发证据链。

## 结论

建议将刘顺 2011 年发表的《三十年来的南京方言研究》编为 `NJH000001`，证据身份标为**研究观点**。它是南京审计学院主办学报刊载的南京方言研究综述，期刊官方站至今公开摘要和 PDF，适合作为读者进入南京方言研究史的第一张“索引地图”。它不是原始语音记录、方言调查数据集或地方志原件，不能替代文中引用的原始研究，也不能证明 2026 年南京话的实际使用情况。[S1][S2][S3]

首篇短专题建议回答：**“一篇综述能告诉我们什么？”** 正文用该论文演示三件事：研究观点必须归属于作者；综述可以提供继续追查的书目线索；综述转述的语音和语言变体结论，在核到原研究以前不能写成本站定论。[S2][S3]

## 可用于首篇专题的事实与边界

### 1. 这篇文章是“研究回顾”，不是南京话本身

**可写：** 作者在摘要中把近三十年的研究分为语音、词汇和语法三个方面，并认为当时研究“还不全面、不系统、不深入”。“引言”说明文章的任务是对既有成果作述评并提出后续方向。[S2, p. 92]

**不可写：** “学界已经完整证明南京话是什么。”作者恰好提醒读者研究仍有明显空白；这篇论文也没有提供可供重新分析的原始录音、逐条调查问卷或完整发音人数据。[S2, pp. 92, 96–97]

### 2. 它能提供下一步查证的书目入口

**可写：** “引言”和参考文献明确列出赵元任《南京音系》（1929）、《江苏省和上海市方言概况》（1960）、《南京方言志》（1993）、《南京方言词典》（1994）、《南京话音档》（1997）及《江苏省志·方言志》（1998）等材料；这些书目构成后续逐件寻找原始出版物的线索。[S2, pp. 92, 97]

**不可写：** 仅凭刘顺的转述，就把上述材料中的调查结论改写成本站的无归属史实。专题若要说明某一具体语音、词汇或语法现象，应继续核对原研究的版本、页码、调查对象、时间和方法。

### 3. 文献本身显示南京话描写并非只有一个答案

**可写：** 刘顺在转述《南京方言志》和《〈南京方言词典〉引论》时，分别记录了 42 个和 49 个韵母的分析；同页还概括了不同研究者对部分单字调调值的分歧。这足以说明“分析框架和材料不同，研究结论可能不同”，网站应并列归属观点，不宜把一个数字包装为唯一标准。[S2, p. 93]

**不可写：** 本次研究没有独立复核两部原著，也没有评判哪一种音系分析更正确。`42` 和 `49` 是刘顺对两项研究的转述，在原著到手前必须保持二级引用身份。[S2, p. 93]

### 4. 文章中的语言变化论断只能按作者观点呈现

**可写：** 论文综述了若干关于入声、轻声、儿化、变调及向普通话靠拢的研究，并在结语提出继续研究新老派差异和形成原因的必要性。[S2, pp. 93–97]

**不可写：** 文章发表于 2011 年，所述调查更早，不能直接证明 2026 年南京主城区的语音分布，也不能从这些研究对象推算南京全体人口。文章中的“新派”“老派”是被引用研究的分类，不是本站对说话者身份或“正宗”程度的判定。[S2, pp. 92–93, 96–97]

## 短专题文章结构建议

标题建议：**《一篇综述能告诉我们什么？》**

1. **先认材料。** 展示论文的作者、原刊、年份、页码和证据身份，不展示受版权保护的扫描页。
2. **再看作者做了什么。** 用摘要中的一句必要短引说明这是一篇研究述评，而不是原始调查。
3. **沿参考文献向前走。** 选三至六项原研究作为后续待核线索；链接到未来档案条目时标清是否已经读到原件。
4. **把分歧保留下来。** 用 `42 / 49` 个韵母的二级引用案例说明“资料有出处”仍不等于“只有一个结论”。
5. **明确停止线。** 不用这篇 2011 年综述断言今日南京话，不借“学者认为”隐藏具体责任者。

专题的关键论断应直接链接 `NJH000001`；条目再提供官方摘要、PDF、完整引用和“能证明 / 不能证明”说明，从而形成“专题文章 → 档案条目 → 来源原页”的第一条完整链路。

## 元数据冲突与处理

1. 当前详情页的通用元标签写作现刊《南京审计大学学报》及现 ISSN `2096-3114`，但 2011 年原刊 PDF 清楚写作《南京审计学院学报》，文章编号前缀为 `1672-8750`。[S1][S2]
2. 官方“本刊介绍”确认现刊原名为《南京审计学院学报》。因此条目应保留**发表时原刊名**，并把现刊名作为机构沿革信息，而不是覆盖历史书目信息。[S4]
3. 官方 XML 导出给出年份 2011、卷 8、期 1 和 ISSN `1672-8750`，但期刊名为占位值 `test`，页码为空；RIS 导出同样缺页码。因此原刊 PDF 是页码、原刊名和文章编号的优先证据。[S2][S5]
4. 官方摘要页未给出 DOI 或 CSTR，故稳定定位使用官方内部文章号、完整书目信息和文章编号，不把普通 URL 冒充 DOI。[S1][S2]
