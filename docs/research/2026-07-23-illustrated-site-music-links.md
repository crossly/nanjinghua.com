# 插画版可链接歌曲小清单

研究日期：2026-07-23

用途：为“南京话小博物馆”的插画版提供可放在故事页、场景卡和页尾的**外链音乐**。这不是南京话歌曲史，也不把“南京题材”一概说成“南京话歌曲”。首发不自托管歌曲、歌词、封面或音频片段。

## 页面编排建议

1. 首页只出现一首：在公交站场景放《挤公交》的“去听原曲”外链，文字用“这一路，很多人还记得”。
2. 人物称呼故事页才放《潘西来了》；不在首页先解释“潘西”，也不把它定义为所有南京人都会说的称呼。
3. 节日 / 夫子庙场景使用《Come on！莱斯狗！》，让音乐成为插画的情绪延伸。
4. “方言改编”单独做一张小卡放《南京Style》或《我的挖掘机（南京话版）》，不要将改编曲与口述记忆、白局或历史资料混为同一类内容。

## 版权与嵌入边界

- 建议只链接至曲目的官方平台页，或在平台明确提供、且其嵌入条款允许时使用官方嵌入播放器。
- 不下载、转码、缓存或自行提供歌曲；不复制完整歌词、专辑封面或节目画面。本站原创说明采用的开放许可不覆盖这些录音及图像。[S6]
- 即使平台显示可以播放，也不等于本站取得同步、复制、改编或再分发许可。每个平台的可播放状态会随地区和版权变化。
- 曲名、演者、专辑与发行信息来自平台目录；除《挤公交》的平台标注外，本清单不把目录数据扩展成语言学鉴定。正式使用前仍应由编辑试听，并为每一首保留官方链接。

## 未采用

- **《这块是南京》**（丁乙宁）：曲目虽明确使用南京方言，但歌词中的地域对立和攻击性语气不适合本版温和、开放的城市记忆定位。
- **《房的一米》**（D-Evil）：有很强的本地生活感，适合日后“住房与城市生活”专题；主题偏成人现实议题，不适合作为首屏或儿童友好插画配乐。
