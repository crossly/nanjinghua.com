# T07 研究记录：今天的南京话怎样变化？

研究日期：2026-07-17

用途：支撑专题“今天的南京话怎样变化？”、`NJH000012`—`NJH000014` 三条首发档案，并保存后续扩展候选。正式结论只采用已经核到原页或正式全文、能够交代样本边界的材料；检索摘要只用于发现候选。

审核状态：吴波、顾黔论文全文和南京大学转载页，以及三份后续英文实验全文及其首发页，均已由编辑核对；研究副本保存在 Git 仓库之外。论文中的音系分析尚未经过南京方言或社会语音学专家复核。

## 结论与停止线

现有公开材料足以说明“当代南京话内部存在可测量差异”，但不足以把差异写成全市人口的单线变化史。首发建立了三条窄证据链：

1. 吴波 2006 年研究留下 39 份南京出生高二学生有效样本，并分别记录问卷、单字音与语流音；不同任务下的分类比例并不相同。[S6, pp. 84–91]
2. 顾黔 2015 年论文报告 2013 年 2—3 月调查的 67 人、四个年龄组和 40 个入声例字；部分项目中青年组比中年组保留更多，拒绝简单的单调年龄叙事。[S7, pp. 385–387]
3. 南京大学 2013 年转载报道保存四个调查片区和家庭、学校、邻里等场景，但没有公开四份问卷的完整样本数、抽样方法或各项分母。[S8]

另有三篇已核正式全文的实验材料，合计覆盖 **年龄、语言经历、使用场景** 三个维度，留作后续扩展。当前材料仍没有一项以可比较样本直接检验南京不同片区：南京大学报道虽然列出四个片区，却缺完整分母；Liu 2023 只笼统称参与者居住在南京南部，既没有出生 / 成长地点，也没有片区对照，不能写成“南城与其他片区不同”。

本站不得把实验中的多数结果改写成“南京人通常这样说”，也不得把未出现于有限刺激表的形式称为“罕见”。凡使用“较多、较少、更敏感、较高”等比较词，必须紧邻标注具体研究、样本组和任务。普通话拼音只能作为检索辅助；三项研究使用 IPA、声学值或论文自己的音系标签时，不能用普通话拼音替代为南京话精确注音。

## 专题写作规则

1. **把维度写在结论里。** 不写“年轻人已经能分清 n/l”，写“在 Johnson、Song 2016 的 16 人年轻组、伪词相似度任务中，`[n]` / `[l]` 的差异评分高于 16 人年长组”。
2. **把成长地点与采集地点分开。** 三篇都没有提供可录入的逐人成长片区；南京师范大学是候选 A 的实验地点，不是参与者成长地。
3. **把实验场景与日常使用分开。** 听辨伪词、按陈述语调读词、在线读设计句都不是自然会话；不能从实验命中率或显著性推断生活频率。
4. **把文献比较与同期对照分开。** 候选 B 借既有文献定位 35–65 岁样本，却没有直接录入 75 岁以上组；专题可以说它提供“当代中年组基线”，不能说它独立证明老派消失。
5. **不用“正宗 / 最地道”认证说话者。** 来源中的 `authentic` 只作为需审查的作者措辞，不进入本站无归属结论。
6. **不用普通话拼音冒充南京话注音。** 搜索页可以保留汉字 / 普通话拼音检索键；正文精确语音只保留来源中的 IPA、F0、调类或经专家复核的南京话标音，并标明体系。
7. **给原创录音留空位而不填充。** 专题预留“本站首批语音样本”关联区，状态写“等待授权材料”；不使用合成音、论文刺激音频或第三方视频替代。
