# 非音频无障碍验收记录

日期：2026-07-18

范围：首页、专题、长题名档案、检索、制度页和线索提交。媒体播放、转写、播放器、R2 媒体及其失败状态不在本次范围内。

## 自动检查

- 使用 `@axe-core/playwright` 的 WCAG 2 A/AA、2.1 A/AA 和 2.2 AA 规则扫描六类代表页面。
- 桌面 Chromium 与 Pixel 7 视口均运行；检查页面稳定后的站内 DOM。
- 每个代表页面同时检查文档横向溢出，长英文与中文题名均纳入。
- 以 `prefers-reduced-motion: reduce` 验证首页入场动画和图标过渡关闭。

## 键盘与语义

- 逐项验证首页品牌、检索入口、返回导航、搜索框、五项筛选和提交按钮的焦点顺序与可见焦点。
- 逐项验证线索表单的类型、说明、可选链接、档案编号、联系字段、数据规则确认框和提交按钮可由键盘到达；完成一条文本线索提交并取得确认编号。
- 人工对照页面可见顺序、标题层级与主要地标，结果如下；这份记录不是屏幕阅读器实测的替代品。

| 页面 | 人工检查结果 |
| --- | --- |
| 首页 | 页头后依次为主标题、开篇集合、历史路径、精选档案和页脚；区块标题与页面顺序一致。 |
| 专题 | 页头后依次为专题题名、署名与审核信息、正文、参考材料和页脚；正文标题没有越级。 |
| 档案 | 页头后依次为档案题名、事实栏、档案说明、正文、核查入口和页脚；永久编号与证据身份先于正文。 |
| 检索 | 页头后依次为检索标题、查询与筛选表单、结果摘要、结果列表和页脚；筛选标签均与控件关联。 |
| 制度 | 页头后依次为制度题名、当前制度正文、制度目录和页脚；当前页面在目录中可辨认。 |
| 线索提交 | 页头后依次为页面题名、信息用途、线索表单和页脚；表单说明先于输入控件。 |
- 1940 年地图有替代文本和来源链接；首页历史路径使用有序列表；扫描档案提供摘要、人工说明、定位信息和来源入口。
- 证据身份、权利状态、空结果、撤回和错误反馈均有可读文字，不以颜色作为唯一信息。
- 页面没有弹层或菜单，因此不存在焦点困住流程。

## 已知边界

- Axe 不代替屏幕阅读器、放大软件或真实用户测试。
- 自动键盘流程以一个可聚焦、可确认并触发 token callback 的测试替身保留 Turnstile 验证步骤；这不等同于独立审计第三方跨域 iframe 的内部实现。
- 中国大陆真实网络与自定义域名可用性属于发布和运维验收，不由本地无障碍自动化证明。
