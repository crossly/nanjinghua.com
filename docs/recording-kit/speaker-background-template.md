# 说话者背景模板

本表用于解释语音样本之间的差异，不用于认定“正宗南京人”。一位参与者填写一次；经历发生变化时另作修订，不覆盖旧表。

## 管理信息

- 参与者代码：`NJH-S__`
- 背景表填写日期：`YYYY-MM-DD`
- 填写方式：□ 参与者自填　□ 采集者根据参与者回答记录

管理信息不公开真实身份。参与者代码与真实身份的对应关系只在受限身份目录保存。

## 必要背景

| 字段 | 填写内容 | 公开前概括规则 |
| --- | --- | --- |
| 出生年代 | □ 1940 年代 □ 1950 年代 □ 1960 年代 □ 1970 年代 □ 1980 年代 □ 1990 年代 □ 2000 年代 | 只保留十年区间，不填写具体年月日 |
| 成长片区 | 主要成长的南京片区；可填旧地名并补充大致当代位置 | 到片区或区级，不写街道门牌、学校或单位 |
| 长期居住 | 曾连续居住约五年以上的城市或片区及大致阶段 | 用城市、片区和概括阶段，不写详细地址 |
| 家庭语言 | 小时候家中常听、常说的语言或地方话，以及主要交谈对象 | 只写语言使用关系，不写亲属姓名 |
| 普通话使用 | 很少／偶尔／经常／几乎总是；可补充上学、工作、家庭等场景 | 只保留频率和场景，不写单位名称 |

## 填写边界

- 不因为口音、籍贯或户籍替参与者推断答案。
- 参与者不确定时写“不确定”或大致范围，不追问精确日期和地址。
- 不增收职业单位、学校名称、身份证件、家庭成员姓名、联系方式、收入、健康、民族或政治信息。
- 公开前由编辑再次概括片区和时间，确认五类背景组合后不会轻易识别个人。
