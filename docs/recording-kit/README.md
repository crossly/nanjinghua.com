# 南京话真人语音采集包

版本：1.0.0  
发布日期：2026-07-17  
适用对象：协助南京话公共数字档案采集语音的非专业采集者

这套材料帮助采集者在常见手机或录音机上记录成年南京话使用者的声音。目标是留下带有背景、授权和来源关系的语言记录，不是评选或表演“最正宗”的南京话。

## 使用边界

- 只接受录音当天已经年满 18 周岁的参与者；不能确认年龄时不要录制。
- 参与完全自愿。先解释用途和三项授权，再录音；参与者可以跳过任何问题或随时停止。
- 公开播放、公开下载和研究复用分别选择，同意一项不代表同意另外两项。
- 真实身份、授权凭证和公开元数据分开保存；文件名和录音开头都不说真实姓名。
- 采集者不通过公开表单、电子邮件或聊天软件发送录音和授权文件。先在网站提交“录音意愿”，等编辑约定安全交接方式。
- 录音不会自动成为正式档案。编辑仍需核对文件、授权、背景和内容。

## 包内文件

1. `prompts.md`：共同词句、自然讲述和访谈提示。
2. `technical-guide.md`：环境、设备、格式、距离、命名和原件保留。
3. `speaker-background-template.md`：只收集解释语言差异所需的背景。
4. `consent-form.md`：成年人确认、三项独立授权和撤回说明。
5. `secure-handoff-checklist.md`：采集前后与项目接收端逐项核对。
6. `rehearsal/`：用虚构身份和静音音频完成的流程演练，不含真人公开素材。
7. `research-notes.md`：制定本包时核查的专业指南与取舍。

## 一次完整采集

1. **先联系**：确认参与者成年且愿意了解项目，不要求当场答应。
2. **分配代码**：使用 `NJH-S01` 一类参与者代码；代码与真实身份的对应表受限保存。
3. **讲清授权**：逐项阅读授权书，让参与者分别勾选；不预先勾选。
4. **填写背景**：只记录出生年代、成长片区、长期居住、家庭语言和普通话使用概况。
5. **现场试录**：录 20 秒，戴耳机或回放检查人声、底噪和削波。
6. **正式录制**：依次完成共同词句、自然讲述和访谈。参与者可以停顿、重说或不回答。
7. **封存原件**：从设备直接导出原始文件，计算 SHA-256，不覆盖、不裁剪、不重命名设备内唯一副本后再处理。
8. **准备交接**：身份、授权、原始录音、公开元数据分别装入清单规定的目录。
9. **安全传送**：通过编辑约定的加密交接方式传送；密码和下载地址不走同一渠道。
10. **双方确认**：接收端验证校验值与授权范围，采集者收到确认并核对备份后才删除临时工作副本。

## 参与和撤回入口

在 [nanjinghua.com/contribute](https://nanjinghua.com/contribute) 选择“录音意愿”联系项目。提出撤回或变更授权时选择“权利请求”或“隐私或安全请求”，说明参与者代码、化名或录音编号即可；不要在公开说明栏填写真实姓名、身份证号或精确住址。
